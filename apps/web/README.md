# ChemEng RR7 Workspace

This project is a React Router v7 Framework Mode application using a Modular Monolith architecture.

## Structure

- **App**: `app/`
  - **Modules**: `app/modules/` (Feature-based modules with automatic route registration)
  - **Layouts**: `app/layouts/`

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

This will start the React Router v7 app.

### Build

Build all packages and the root app:

```bash
npm run build
```
