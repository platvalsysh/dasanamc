"평생을 함께 지키는 중증·노령 내과 케어"

​

내과 진료는 단순히 질환을 치료하는 것을 넘어,

아이의 ‘삶의 질’을 관리하는 과정입니다.

저는 중증 질환과 노령 반려동물의 건강 관리에

집중하여, 오랜 경험과 데이터를 토대로

최적의 치료 방향을 제시합니다.

​

보호자분의 이야기를 끝까지 경청하며,

아이가 남은 생을 건강하고 편안하게

보낼 수 있도록 최선을 다하겠습니다.

​

​

​

모바일에서 화면 터치 시 바로 전화연결 가능합니다.

반려동물의 평생 건강 관리를 책임지겠습니다.
