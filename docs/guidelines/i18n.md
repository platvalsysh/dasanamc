# 국제화 (i18n)

> **현재 상태**: 단순 i18n 래퍼 도입 완료 ([packages/core/src/ui/i18n.ts](../../packages/core/src/ui/i18n.ts)). 기본 한국어, 공용 키 사전 운영 중. 모듈별 사전 등록은 `registerDict()` 사용. 본격적인 다국어 운영(언어 자동 감지·라우트 prefix 등)은 추후 i18next 마이그레이션 시.

## 사용

```tsx
import { t, setLang, registerDict } from "@repo/core/ui";

<button>{t("common.save")}</button>          // "저장"
<button>{t("common.cancel")}</button>        // "취소"
<p>{t("greeting", { name: user.name })}</p>  // 변수 치환
```

언어 전환 (클라이언트):

```ts
setLang("en");
```

⚠️ **`setLang`은 모듈 전역 상태**라 서버 라우트에서 호출하면 다른 요청에 영향을 미친다. 서버에서 사용자 언어를 사용해야 하면 명시적 인자로 전달:

```ts
t("common.save", undefined, userLang); // 현재 전역 언어를 무시하고 userLang 사용
```

## 모듈별 사전 등록

```ts
// packages/module-board/src/i18n.ts
import { registerDict } from "@repo/core/ui";
import boardKo from "./locales/ko.json";
import boardEn from "./locales/en.json";

registerDict("ko", boardKo);
registerDict("en", boardEn);
```

모듈 부팅 진입점 (route module의 top-level 또는 module.server.ts) 에서 1회 import.

## 사전 파일 구조

```
packages/core/src/ui/locales/
  ko.json   # 공용 키 ("common.save", "common.cancel" ...)
  en.json
packages/{모듈}/src/locales/
  ko.json   # 모듈 특화 키 ("board.list.title" ...)
  en.json
```

## 향후 i18next 로 이전이 필요해질 때

- **언어 자동 감지** (Accept-Language, 쿠키, URL prefix)
- **복수형/성별 변형** (영어 "1 item" / "2 items")
- **컨텍스트별 번역** (같은 "Save" 가 화면마다 다른 의미)
- **번역 관리 도구 연동** (Crowdin, Lokalise, ...)

이 중 하나라도 필요하면 i18next 로 이전.

## 마이그레이션 전략 (점진적)

1. 새로 작성하는 텍스트부터 `t()` 래핑
2. PR 단위로 1-2개 페이지씩 변환
3. ESLint 룰로 신규 하드코딩 차단 (warn 으로 시작):
   ```js
   "react/jsx-no-literals": ["warn", { ignore: [...] }]
   ```
4. 모든 페이지 변환 완료 시 룰을 `error` 로 승격

## 안티 패턴

- ❌ **DB 컬럼에 `name_ko`, `name_en` 분리** — 언어 추가 시 스키마 폭증. JSON 컬럼 + locale key 사용 권장
- ❌ **한 컴포넌트에 한국어/영어 혼재** — 빌더 메뉴 같은 사례 (현재 [packages/admin/src/routes/system/settings/menu.tsx](../../packages/admin/src/routes/system/settings/menu.tsx) 의 "Cancel"/"Save Changes" 와 "메뉴 항목 추가" 공존)
- ❌ **날짜/숫자 직접 포맷팅** — `Intl.DateTimeFormat` / `Intl.NumberFormat` 사용
- ❌ **메일/SMS 템플릿을 코드에 인라인** — DB 기반 템플릿 + 변수 치환
